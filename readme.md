##Read me!
This is template for apps with:

* Spring Boot
* Spring REST
* Spring MVC
* Spring Data JPA
* Spring Security 
* Liquibase
* HSQL DB Embedded
* Log4j2
* Hibernate
* jUnit4

####Got it

`mvn clean package`

You can start it by command

`java -jar [appname].war`

By default it starts on localhost:80
